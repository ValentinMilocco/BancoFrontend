# BancoFrontend

Aplicacion web bancaria desarrollada con React, complementada por un backend separado en Spring Boot, trabajada de forma colaborativa para practicar desarrollo full stack.

## Estado de este repositorio

Este repositorio es un fork con fines de portfolio profesional.

Proyecto original (frontend): https://github.com/BenjaminPeres596/BancoFrontendDesarrollo

Proyecto complementario (backend): https://github.com/BenjaminPeres596/BancoBackendDesarrollo

## Mi contribucion en el proyecto original

Participe como contributor en mejoras funcionales y de calidad, principalmente en:

- Desarrollo y ajustes de pantallas del frontend bancario
- Integracion de servicios con endpoints del backend
- Mejora de experiencia de usuario en flujos de login, movimientos y transferencias

## Convencion de commits

Formato:

`[type]: mensaje-commit`

Tipos:

- `fix`: correccion de bugs
- `feat`: nueva funcionalidad
- `refactor`: refactorizacion de codigo
- `style`: cambios de estilos
- `docs`: documentacion

## Dependencias y ejecucion local

Dependencias principales:

- Node.js 18+
- npm 9+
- React

Pasos:

Instalar dependencias:

```bash
npm install
```

Iniciar la aplicacion frontend:

```bash
npm start
```

La aplicacion se ejecuta en:

- http://localhost:3000

Para entorno completo, levantar tambien el backend complementario:

- https://github.com/BenjaminPeres596/BancoBackendDesarrollo
